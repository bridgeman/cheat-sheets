# Proton Mail Keyboard Shortcuts

Basic navigation and actions remain available regardless of keyboard shortcuts being active or not in the settings.

## Basic navigation

- Move up ↑
- Jump to first Ctrl + ↑
- Move down ↓
- Jump to last Ctrl + ↓
- Move right / expand →
- Move left / collapse ←

## Basic actions

- Apply / open Enter
- Cancel / close Escape
- Open this modal ?
- Select / unselect Space
- Open command panel Ctrl + K
